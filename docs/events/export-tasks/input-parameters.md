# 🔽**Pre/Post Export Generation Task Script Input Parameters**

## Input Parameters

The following table lists Input Parameters that are available for Export Task Logic Scripts.



| Input Parameter | Description |
| --- | --- |
| g_user_id | Current logged User ID |
| g_lb_script_name | Logic Script Name being executed |
| g_exp_config_name | Export Name |
| g_exp_id | Export Execution ID |
