# 🔽**On Submit Workflow Input Parameters**

## Input Parameters

The following table lists Input Parameters that are available for On Submit Workflow Logic Scripts.



| Input Parameter | Description |
| --- | --- |
| g_user_id | Current logged User ID |
| g_lb_script_name | Logic Script Name being executed |
| g_request_id | Request ID |
| g_wf_code | Workflow Code |
