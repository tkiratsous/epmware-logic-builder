# 🔽**Workflow Custom Task Script Input Parameters**

## Input Parameters

The following table lists Input Parameters that are available for Workflow Custom Task Logic Scripts.



| Input Parameter | Description |
| --- | --- |
| g_user_id | Current logged User ID |
| g_lb_script_name | Logic Script Name being executed |
| g_request_id | Request ID |
| g_wf_code | Workflow Code |
| g_wf_stage_name | Workflow Stage Name |
| g_wf_stage_task_id | Workflow Stage Task Internal ID |
