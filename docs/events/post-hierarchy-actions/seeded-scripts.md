# **Seeded Hierarchy Action Scripts**

EPMware provides pre-built (seeded) Logic Scripts that handle common hierarchy management scenarios. 

***Seeded Script : EW_HFM_ENT_SEC_CLASS_AUTOMATION*** <br/>
This Logic Script is provided for all clients. It is mainly used by HFM Applications and for the Entity Dimension. The purpose of this Logic Script is to automatically create a new member in the “Security Class” dimension whenever a new Member in the Entity dimension is created.

Note: The script is available as a reference only. Please copy and modify it for your own use case and criteria.
