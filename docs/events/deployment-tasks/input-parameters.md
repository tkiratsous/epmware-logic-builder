# 🔽**Pre/Post Deployment Scripts Input Parameters**

## Input Parameters

The following table lists Input Parameters :



| Input Parameter | Description |
| --- | --- |
| g_user_id | Current logged User ID |
| g_lb_script_name | Logic Script Name being executed |
| g_dep_config_name | Deployment Configuration Name |
| g_deployment_id | Deployment ID |
| g_dep_scheduled_by_user_name | User who scheduled Deployment |
| g_dep_configured_by_user_name | User who configured the Deployment Manager |
| g_dep_request_id_list | PL/SQL table list of Request IDs which are part the Deployment Execution |
| g_dep_scheduled_by_user_id | User ID who scheduled Deployment |
| g_dep_configured_by_user_id | User ID who configured Deployment |
