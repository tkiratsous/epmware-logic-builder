# 🔽**ERP Interface Scripts Input Parameters**

## Input Parameters

The following table lists Input Parameters that are available for ERP Interface Scripts.



| Input Parameter | Description |
| --- | --- |
| g_user_id | Current logged User ID |
| g_lb_script_name | Logic Script Name being executed |
| g_if_config_name | ERP Import Configuration Name |
| g_app_name | Application Name |
| g_dim_name | Dimension Name |
| g_app_dimension_id | Dimension ID |
