# :material-swap-horizontal:{ .lg .middle } **Property Mapping**

Property Mapping is used to map property values between two or more dimensions for a mapped member and mapped property.<br/>
Like Dimension mapping, it provides two options: Sync or Smart Sync. The third option is a Logic Script (Type: Property Mapping).

Property Mapping helps  :

 -  Copy properties as-is between applications
 -  Modify property values during synchronization
 -  Map properties based on business rules
