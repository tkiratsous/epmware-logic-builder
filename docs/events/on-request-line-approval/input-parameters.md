# 🔽**On Request Line Approval Task Script Input Parameters**

## Input Parameters

The following table lists Input Parameters that are available :



| Input Parameter | Description |
| --- | --- |
| g_user_id | Current logged User ID |
| g_lb_script_name | Logic Script Name being executed |
| g_request_id | Request ID |
| g_request_line_id | Request Line ID |
| g_wf_code | Workflow Code |
| g_wf_stage_name | Workflow Stage Name |
