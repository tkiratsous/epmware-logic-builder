# :material-check-circle-outline:{ .lg .middle } **On Request Line Approval Task Script**


These scripts are assigned to Workflow Review or Approve tasks and are executed when the user approves a line or mass approves lines in a Request. 

This event is useful especially to validate properties or actions taken in a request before it gets approved by the Reviewer or Approver.


These scripts are associated in the Workflow -> Tasks -> Reviewer/Approver type tasks
