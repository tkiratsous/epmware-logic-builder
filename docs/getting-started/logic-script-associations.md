# :material-link-variant:{ .lg .middle } **Logic Script Associations**

The following table describes Logic Script Types and its corresponding configurations required before they can be executed.


| # | Script Type | Associated Configuration Menu | Field |
| --- | --- | --- | --- |
| 1 | Dimension Mapping | Configuration -> Dimension -> Mapping | Mapping Option |
| 2 | Property Mapping | Configuration -> Property -> Mapping | Mapping Option |
| 3 | Pre Hierarchy Actions | Configuration -> Dimension -> Hierarchy Actions | Pre Action Script Name |
| 4 | Post Hierarchy Actions | Configuration -> Dimension -> Hierarchy Actions | Post Action Script Name |
| 5 | Property Derivations | Configuration -> Property -> Derivations | Script Name |
| 6 | Property Validations | Configuration -> Property -> Validations | Script Name |
| 7 | On Submit Workflow | Workflow -> Builder | On Submit Logic Script |
| 8 | On Request Line Approval | Workflow -> Tasks | Validation Script |
| 9 | Workflow Custom Task | Workflow -> Tasks | Custom Function |
| 10 | Deployment Tasks | Deployment Configuration | Pre (Post) Deployment Script |
| 11 | ERP Interface Tasks | ERP Import -> Builder | Pre (Post) Execution Script |
| 12 | Pre Export Generation Tasks |  Export | Pre Export Logic Script |
| 13 | Post Export Generation Tasks | Export | Post Export Logic Script |
